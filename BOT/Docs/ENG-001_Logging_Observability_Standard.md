
# Logging & Observability Standard
**Status:** ✅ Regenerated

Defines structured logging, log levels, correlation IDs, trace IDs, metrics, distributed tracing, alerting, retention, dashboards, and observability standards for every Trading OS service.

## Standards
- JSON structured logs
- UTC timestamps
- Correlation ID on every request
- Trace ID across services
- Metrics first, logs second
- Immutable audit logs
- PII never logged

## Log Levels
TRACE • DEBUG • INFO • WARN • ERROR • FATAL
