# Purpose

Defines the responsibilities and operational procedures for Trading OS administrators.

# Responsibilities

- User management
- Role & permission management
- Capability Registry administration
- Broker configuration
- Market data provider management
- System monitoring
- Backup verification
- Incident response
- Release approvals

# Daily Checklist

- Review health dashboards
- Review incidents
- Verify broker connectivity
- Verify market data providers
- Verify backups
- Review security alerts

# Weekly Tasks

- Audit users
- Review permissions
- Validate capacity
- Review performance
- Test restores

# Emergency Actions

- Activate Kill Switch
- Disable brokers
- Enter maintenance mode
- Restore backups
- Coordinate incident response
